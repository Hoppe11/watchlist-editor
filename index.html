Erstmalige Einrichtung

Dein bisheriges Token wurde gefunden und unten vorausgefüllt. Leg jetzt zusätzlich ein Passwort fest, damit es ab sofort verschlüsselt gespeichert wird.

Das Token wird mit deinem Passwort verschlüsselt und nur auf diesem Gerät gespeichert (AES-256, aus dem Passwort abgeleitet). Ohne Passwort und ohne die Antwort auf die Sicherheitsfrage ist das gespeicherte Token unlesbar – auch für diese App.

Marken-Regel: Ticker ohne Suffix (US-Börse) → Marke in USD; Ticker mit Suffix (.DE, .AS, .L …) → Marke in EUR. Dezimal: Punkt oder Komma, beides ok.

Schnell-Übernahme

Eine Zeile pro Aktie: Ticker: Marke=Wert Marke=Wert …
Gültige Marken: Kauf1, Kauf2, Stopp, Ziel1, Ziel2. Beispiel:
INIO: Kauf1=29.50 Kauf2=26.00 Ziel1=42.00
